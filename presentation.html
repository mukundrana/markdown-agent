🔧 Technical Questions
How does the markdown-based system work?
The system uses pure markdown files to define agent behaviors, workflows, and protocols. Each agent is a markdown file that AI tools can natively read and understand. When you trigger the system with "go baby go", it reads these files and coordinates agents through an 11-stage workflow - all without any compilation or runtime dependencies.
What AI platforms does it support?
It's universally compatible with 17+ AI platforms including Claude Code, Google Gemini, GitHub Copilot, OpenAI Codex/GPT, Alibaba Qwen, DeepSeek, Mistral AI, Cursor IDE, Windsurf IDE, and more. If your AI tool can read markdown, it can use this system.
Can I add my own custom agents?
Absolutely! The system is designed to be extensible. Simply create a new markdown file in the agents/ folder following the template, define the agent's personality and instructions, and it becomes part of your workflow. No coding required - just write in markdown.
How does state management work?
State is managed through JSON files in the session/ folder. The system tracks which stage you're in, what's completed, and can resume from checkpoints. This means you can pause work and continue later - nothing is lost. Plus, there's a visual dashboard (no server needed) to track progress.
💼 Business & Value
What makes this different from existing tools?
Three key differentiators:
1. AI-agnostic - Works with any AI tool, no vendor lock-in
2. Zero dependencies - Pure markdown, no runtime, no compilation
3. Transparent - Every step is readable in markdown, not a black box

Unlike tools that require specific platforms or have opaque systems, ours is open, portable, and future-proof.
What's the learning curve?
Under 1 hour to be productive! Installation takes seconds - just copy the folder and run install.js. To use it, say "go baby go" and describe what you want. The agents handle the rest. For customizing agents, basic markdown knowledge is all you need.
Can this work with existing projects?
Yes! The markdown-agent/ folder is self-contained and doesn't interfere with your existing codebase. Simply copy it into any project and you're ready. It works with any programming language, framework, or project structure.
🚀 Practical Questions
How long does it take to set up?
Under 2 minutes. Three commands: copy folder, navigate to it, run install.js. That's it. No API keys to configure, no servers to set up, no dependencies to install. It just works.
What happens if an agent makes a mistake?
The system has built-in verification and quality checks. Multiple agents review each other's work - the Tester agent is specifically skeptical and looks for bugs. The SECURITY agent scans for vulnerabilities. If issues are found, they're flagged before deployment. Plus, the checkpoint system lets you roll back to any stage.
Is this open source?
Yes! The entire system is open source. You can fork it, customize it, contribute agents back to the community, and even create your own agent marketplace. We believe in transparent, collaborative AI tooling.
